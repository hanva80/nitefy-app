# First User Test Script

Use this script with 3-5 early testers.

## Context

You are going out tonight with friends in Düsseldorf and nobody knows where to go.

## Tasks

1. Open NITEFY.
2. Set your music preference.
3. Pick the vibe you want tonight.
4. Set your budget and dress code comfort.
5. Choose a recommendation.
6. Open the venue detail page.

## Questions

1. What do you think NITEFY does?
2. Why do you think the first venue was recommended?
3. What information helped you decide?
4. What information was missing?
5. Would you use this before going out tonight?

## What To Watch

- Do users understand the product in under 10 seconds?
- Do they trust the match reasons?
- Do visuals help them understand the vibe?
- Does location feel clear enough?
- Are the cards too detailed or too light?
