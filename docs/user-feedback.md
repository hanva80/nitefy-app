# User Feedback

This document collects early qualitative feedback from test users.

## Persona 1

**Profile:** Female, 21 years old

### Key Feedback

- The product is useful for finding cool party spots based on personal preference.
- It would be helpful when going out with a friend group and nobody knows where to go.
- Location accuracy may need improvement or clearer communication.
- The design and available options were positively received.

### Product Insight

The strongest use case is last-minute decision-making for friend groups. The location experience must feel reliable because inaccurate location can reduce trust immediately.

## Persona 2

**Profile:** Female, 28 years old

### Key Feedback

- First impression was positive because of the simplicity of the app.
- The most important information felt easy to find.
- The product could be useful for last-minute nightlife decisions.
- International availability would make the concept stronger.
- Location-based usage is important.
- Missing features include stronger recommendations and ratings.
- Stronger visuals are needed to understand what to expect.
- Dress code and atmosphere preview are important.

### Product Insight

The interface should stay simple, but recommendations need to feel more trustworthy. Users want to preview the vibe before choosing a venue.

## Main Learnings

1. The product should focus on places, not on finding people.
2. Last-minute discovery is a strong use case.
3. Friend-group indecision is an important problem.
4. Visual previews are essential.
5. Ratings and recommendation logic need to be clear.
6. Location accuracy is critical for trust.
7. Dress code and expectations should be included early.
8. International expansion is attractive, but not part of the first MVP.
