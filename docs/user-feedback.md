# User Feedback

## Main Learnings

1. The product should focus on places, not on finding people.
2. Last-minute discovery is a strong use case.
3. Friend-group indecision is an important problem.
4. Visual previews are essential.
5. Ratings and recommendation logic need to be clear.
6. Location accuracy is critical for trust.
7. Dress code and expectations should be included early.
8. International expansion is attractive, but not part of the first MVP.
