# MVP Scope

## MVP Goal

Validate whether users understand and value a nightlife recommendation product that suggests places based on personal preferences, location, music taste, and vibe.

## Core MVP Features

### 1. Music Taste Analysis

NITEFY should make music taste feel like a primary input. The MVP can start with demo Spotify and Apple Music analysis profiles that map listening taste to music styles and nightlife vibe.

The first validation version can simulate this analysis before adding real OAuth and API integrations.

The simulated scan should still feel believable. It should show:

- The selected provider
- A short scanning state
- Recent listening window
- Top genres
- Top artist signals
- Confidence score
- How the music taste changed the recommendations

### 2. Quick Preference Profile

The user answers a short set of questions to define what kind of night they are looking for.

Possible inputs:

- Music preference
- Preferred atmosphere
- Budget range
- Distance preference
- Dress code comfort
- Type of venue
- Group or solo context

### 3. Location-based Recommendations

The product recommends nightlife spots within a practical radius, initially around 15 km.

### 4. Venue Cards

Each recommendation should show the most important information quickly.

Recommended information:

- Venue name
- Type of place
- Music style
- Location
- Distance
- Price level
- Dress code
- Opening hours
- Rating
- Visual preview
- Short vibe description
- Why this place matches the user

### 5. Strong Visual Preview

Users should understand what to expect before choosing a place. Visuals should communicate atmosphere, crowd, dress code, and energy.

### 6. Recommendation and Rating System

Ratings and recommendations are important for trust. The first version can start simple, but the product should clearly show why a place is recommended.

## Out of Scope for First MVP

These features are not priority for the first version:

- Direct ticket sales
- Complex group matching
- Full social network features
- Chat between users
- Advanced event organizer tools
- International expansion
- Complex AI automation
- Full Spotify or Apple Music account sync

## Success Criteria

The MVP is successful if test users can quickly understand:

1. What the product does
2. Why a venue is recommended
3. Whether music taste analysis feels useful
4. Whether they would use it before going out
5. What information is missing before making a decision
